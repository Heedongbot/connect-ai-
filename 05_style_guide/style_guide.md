# NutriStack Lab: Master Style Guide

마스터님의 고품질 블로그 포스팅 스타일을 복제하기 위한 핵심 규칙입니다. 모든 에이전트는 이 가이드를 엄격히 준수하십시오.

## 1. 문체 및 톤 (Tone of Voice)
- **전문가 동네 형 (Expert Big Brother)**: 지식은 압도적인 수석 연구원 급이지만, 말투는 친근한 동네 형처럼 합니다. (~해요 체를 기본으로 하되 대화하듯 편안한 구어체 사용)
- **딱딱함 제거**: "반드시 수행해야 한다" 대신 "이건 꼭 챙겨야 해, 왜냐하면..." 식으로 독자의 눈높이에서 친절하게 설명합니다.
- **직관적 비유**: 어려운 과학 용어를 설명할 때는 일상적인 비유(예: 엔진 오일, 배터리 등)를 적절히 섞어 지루함을 덜어냅니다.

## 2. HTML 구조 및 디자인 가이드
- **도입부 요약**: <h1> 태그 직후에 `<div style="background-color: #f8f9fa; ...">` 태그를 사용하여 2-3문장의 핵심 요약을 작성하십시오. (Courier New 폰트 사용)
- **Key Takeaways**: 본문 시작 전 `<div style="background: rgb(240, 247, 255); border-left: 4px solid rgb(42, 100, 150); ...">` 박스를 사용하여 핵심 요점을 리스트로 정리하십시오.
- **테이블(Table)**: 데이터 비교 시 반드시 `<thead style="background: rgb(42, 100, 150); color: white;">`를 가진 테이블을 사용하십시오.
- **이미지 캡션**: 이미지 아래에 `<span style="background-color: #f8f9fa; ...">` 태그를 사용하여 Courier New 폰트로 설명을 덧붙이십시오.
- **관련 링크**: 섹션 끝에 `→ <a href="...">Related: ...</a>` 형식으로 내부 링크를 배치하십시오.

## 3. SEO 및 메타 데이터
- **검색 설명(Meta Description)**: 글 상단에 별도로 표기하여 마스터님이 복사하기 쉽게 제공하십시오.
- **JSON-LD**: 글 하단에 FAQPage 타입의 구조화된 데이터(JSON-LD)를 반드시 포함하십시오.
- **FAQ**: 독자들이 궁금해할 만한 질문 5개를 선정하여 h3 태그와 함께 FAQ 섹션을 만드십시오.

## 4. 금지 사항
- "안녕하세요", "반갑습니다" 같은 상투적인 블로거식 인사를 절대 하지 마십시오.
- 잡담 없이 바로 본론(Mørketid 같은 핵심 개념)으로 들어가십시오.
